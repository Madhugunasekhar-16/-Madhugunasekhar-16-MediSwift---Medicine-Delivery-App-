# MediSwift - Medicine Delivery App

A full-stack MERN application for ordering medicines online with admin management.

## Features

### User
- User registration & login
- Browse medicines
- Add medicines to cart
- Upload prescription if required
- Place orders
- View order history
- Update profile
- Change password

### Admin
- Admin dashboard with analytics
- Manage medicines
- View all orders
- Manage users

### Security
- Protected routes
- Admin route protection
- Token expiration handling
- API error handling

## Tech Stack

Frontend:
- React
- Tailwind CSS
- React Router

Backend:
- Node.js
- Express.js
- MongoDB
- JWT Authentication
